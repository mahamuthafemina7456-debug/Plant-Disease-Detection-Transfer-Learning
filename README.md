# Plant-Disease-Detection-Transfer-Learning
Transfer Learning based plant disease detection system using ResNet50 to classify tomato leaf images into Healthy, Early Blight, and Late Blight categories with high accuracy.

# Plant Disease Detection using Transfer Learning

This project uses ResNet50 pretrained CNN model to classify tomato leaf diseases.

Classes:
- Healthy
- Early Blight
- Late Blight

Tools Used:
- Python
- TensorFlow
- Google Colab

Accuracy: 96.5%
